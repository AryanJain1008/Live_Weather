# Live Weather Application

## Overview

The Live Weather Application is a web-based project developed using HTML, CSS, and JavaScript. It provides users with real-time weather information for any city by integrating a live weather API, ensuring accurate and up-to-date data.

---

## Features

* Search weather details by city name
* Real-time weather data using API integration
* Display of current temperature
* Humidity level indication
* Wind speed information
* Dynamic weather icons based on conditions
* Responsive and user-friendly interface

---

## Technology Stack

* HTML
* CSS
* JavaScript
* Weather API (for real-time data)

## Installation and Setup

To run this project locally, follow these steps:

1. Clone the repository:

   ```
   git clone https://github.com/your-username/Live_Weather.git
   ```

2. Navigate to the project directory.

3. Open the `index.html` file in any web browser.

---

## API Integration

This application uses a real-time weather API to fetch live weather data based on user input. The API provides information such as temperature, humidity, wind speed, and weather conditions.
